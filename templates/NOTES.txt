== Runner Status ==

Check StatefulSet status:
  kubectl get statefulset {{ include "actions-runner.fullname" $ }}{{ if $.Values.includeNamespace }} -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}

View runner logs:
  kubectl logs statefulset/{{ include "actions-runner.fullname" $ }} -c runner{{ if $.Values.includeNamespace }} -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}

Watch all runner pods:
  kubectl get pods -l app.kubernetes.io/instance={{ .Release.Name }}{{ if $.Values.includeNamespace }} -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}

== Re-registering Runners ==

If a runner loses its GitHub registration (token expired or runner removed), delete
its credentials PVC to force re-registration on the next pod start:

  # Replace <N> with the pod index (0, 1, 2, ...)
  kubectl delete pvc runner-creds-{{ include "actions-runner.fullname" $ }}-<N>{{ if $.Values.includeNamespace }} -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}
  kubectl delete pod {{ include "actions-runner.fullname" $ }}-<N>{{ if $.Values.includeNamespace }} -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}

The init container will automatically re-run config.sh with the token from the Secret.
Make sure the Secret contains a fresh registration token before deleting the PVC.
