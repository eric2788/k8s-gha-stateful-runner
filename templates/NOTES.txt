== Runner Status ==

Check StatefulSet status:
  kubectl get statefulset {{ include "actions-runner.fullname" $ }}{{ if $.Values.includeNamespace }} -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}

View runner logs:
  kubectl logs statefulset/{{ include "actions-runner.fullname" $ }} -c runner{{ if $.Values.includeNamespace }} -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}

Watch all runner pods:
  kubectl get pods -l app.kubernetes.io/instance={{ .Release.Name }}{{ if $.Values.includeNamespace }} -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}

== Re-registering Runners ==

If a runner loses its GitHub registration (runner removed from GitHub), re-register it:

  Step 1 — refresh the Secret with a NEW registration token FIRST
  (GitHub tokens expire ~1 hour after generation):
  kubectl create secret generic {{ include "actions-runner.secretName" $ }} \
    --from-literal=ui_token=NEW_REGISTRATION_TOKEN \
    --dry-run=client -o yaml | kubectl apply -f -{{ if $.Values.includeNamespace }} \
    -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}

  Step 2 — delete the credential PVC and pod (replace <N> with pod index 0, 1, 2, ...):
  kubectl delete pvc runner-creds-{{ include "actions-runner.fullname" $ }}-<N>{{ if $.Values.includeNamespace }} -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}
  kubectl delete pod {{ include "actions-runner.fullname" $ }}-<N>{{ if $.Values.includeNamespace }} -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}

The StatefulSet recreates the pod. Because the credential PVC was deleted, the init
container runs config.sh with the new token and saves fresh credentials.

Note: scaling the StatefulSet down and back up retains the credential PVCs so runners
reconnect without re-registration. Only delete a PVC when you explicitly need to
force re-registration for a specific runner.

== Uninstalling ==

  helm uninstall {{ .Release.Name }}{{ if $.Values.includeNamespace }} -n {{ default .Release.Namespace $.Values.namespaceOverride }}{{ end }}

All credential PVCs are deleted automatically (whenDeleted: Delete).
Runner entries in GitHub Settings → Actions → Runners will remain as offline.
Remove them manually from the GitHub UI or via the API.
